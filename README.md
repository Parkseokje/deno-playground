## To run source code

- Deno Installed
- Execute
  ```bash
  $ deno run samples/hello.ts
  $ deno run samples/datetime.ts
  $ deno run --allow-write samples/createFile.ts
  $ deno run --allow-read samples/readFile.ts
  $ deno run --allow-net samples/simpleServer.ts
  ```
